# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn data to identify patterns, trends, and business insights related to customer retention and customer behavior.

The project was developed using SQL, Python, and Tableau to perform data analysis, visualization, and dashboard creation.

---

# Objectives
- Analyze customer churn behavior
- Identify factors affecting churn
- Study customer contracts and payment methods
- Analyze monthly charges and tenure
- Build an interactive churn dashboard
- Generate business insights for customer retention

---

# Tools & Technologies Used
- Python
- SQL
- Tableau
- pandas
- matplotlib
- MySQL

# Key Insights
- Month-to-month contracts have the highest churn rate.
- Customers with higher monthly charges churn more frequently.
- Fiber optic users show higher churn behavior.
- Long-term customers are less likely to churn.
- Electronic check users have higher churn rates.
